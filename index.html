access - a - read

A jQuery plugin to make articles on your website easy to read and distraction free.

view demo download

access - a - read

How to Use:

  1. Download jQuery plugin here.
  2. Link accessStyle.css stylesheet in the head of your HTML document.

  3. Add a div with a class of clearReadSwitch at the bottom of your HTML document before the closing body tag. This div holds the icon your user will click to bring up the access-a-read page.

    <div class="clearReadSwitch"></div>

  4. Link jQuery and the main.js file at the bottom of your HTML document in a script tag

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

<script src="clearRead.js"></script>
  5. In the stylesheet, the clearReadSwitch has been position fixed at the bottom of the page. The icon will then be added to your page automatically!

That's it! Your user will now be able to read the articles you've posted with ease and distraction free.
